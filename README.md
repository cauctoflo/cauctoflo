## 🧑‍💻 À propos de moi

Je suis un développeur **full‑stack** passionné d'informatique depuis mes 14 ans. J'aime concevoir des applications modernes (Laravel, Next.js, TailwindCSS) mais aussi mettre les mains dans l'**infrastructure** : serveurs Linux, Docker, clusters haute dispo. Mon parcours a commencé sur Minecraft (Skript, Spigot, Paper), puis s'est étendu au web et au DevOps.

- 🎓 **Formation :** Licence Informatique — UFR MIM, Université de Lorraine
- 🌐 **Portfolio :** [florentindev.eu](https://florentindev.eu)
- 📍 **Localisation :** Metz, Grand Est, France
